== Installation ==

So installieren Sie das used-design Wordpress PlugIn

1. Kopieren Sie den gesamten Pluginordner in Ihr `/wp-content/plugins/` Verzeichnis
1. Aktivieren Sie das used-design Plugin im 'Plugins' menu von WordPress
1. Folgen Sie der Anleitung im "Settings / used-design" menu von WordPress
